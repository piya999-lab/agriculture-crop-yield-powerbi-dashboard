## Project Overview
This project presents an interactive Power BI dashboard analyzing agricultural crop yield and production trends across Indian states.

Dataset Source:  
[https://www.kaggle.com/datasets/akshatgupta7/crop-yield-in-indian-states-dataset](https://www.kaggle.com/datasets/akshatgupta7/crop-yield-in-indian-states-dataset)

---

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Cleaning
- Data Visualization

---

## Key Features
- State-wise crop production comparison
- Rainfall vs yield trend analysis
- Interactive slicers and filters
- KPI indicators (Total Years, States, Crops)
- Insight-driven visual storytelling
